# Code Review Assistant

AI-powered code review tool that analyzes code quality, bugs, and best practices.

## Features
- Upload code files or paste code
- AI-powered analysis
- Detailed review reports
- Download reports

## How to Use
1. Upload your code file or paste code
2. Select programming language
3. Click "Analyze Code"
4. View and download the review report
